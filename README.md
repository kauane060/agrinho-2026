##  Sobre o Projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026 com o objetivo de demonstrar como a tecnologia pode contribuir para uma agricultura mais eficiente e sustentável.

A proposta apresenta a Agricultura de Precisão como uma solução para reduzir o desperdício de água e fertilizantes, utilizando sensores, Internet das Coisas (IoT), Inteligência Artificial e programação para auxiliar na tomada de decisões no campo.

Além disso, o projeto possui um simulador interativo que demonstra como o uso dessas tecnologias pode contribuir para a economia de recursos naturais.

---

##  Objetivos

- Demonstrar a importância da Agricultura de Precisão.
- Incentivar o uso consciente da água.
- Explicar como a matemática auxilia nos cálculos agrícolas.
- Mostrar a aplicação da programação no campo.
- Apresentar tecnologias como IoT e Inteligência Artificial.
- Promover práticas sustentáveis na agricultura.
